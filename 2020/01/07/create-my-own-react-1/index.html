简易react实现(1)

Posted on 2020-01-07 | In React   |   字数统计: 1.9k(字)   |   阅读时长: 7(分) | Visitors

前言

  • 该系列内容主要是记录实现一个简易react实现的过程和思维，从而加深对react本身的理解。

本篇是实现简易react的第一篇，这是个一步步实现的过程，实现的过程就是发现问题和解决问题，每个实现的功能点也是由浅入深。并且注重的是功能实现，对于性能不多考虑。我把我的这一个小工具称之为Fishtail。使用的语法为TypeScript，使代码阅读起来更容易理解。
本篇实现的步骤主要有：

  1. ReactDOM.render基本功能，使用该方法，把一个React Element元素渲染到目标元素中。
  2. React.createElement方法，该方法利用JSX解析或传入特定格式的数据结构，生成一个React Element。实现这个方法就可以衔接render方法实现JSX写法渲染元素。
  3. 实现基本Virtual DOM效果，利用render方法和JSX实现重(chóng)用DOM更新。

tips: 由于使用TypeScript编写的代码，我配置了rollup进行了代码编译和打包。如果想测试代码，可以选择：

  1. JS实现，顺便练练手；
  2. TS实现，自己配置一下打包，再次练练手；
